Proposed metro stations in Belgrade and accessibility area

About this map

This map shows proposed metro stations in Belgrade, along with their accessibility areas (5, 10, and 15 minutes). Additionaly, each area contains population count (click on the station and see).

The station locations are taken from the Development plans as of January 2025.

The map was created by Semyon Skotnikov using Daniel Gorokhov's Python and PostGIS code.
The population data source is GHSL.
